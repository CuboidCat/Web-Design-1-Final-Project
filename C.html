The History of C

1978 The "K&R" standard, the first widely available description of C, was published.
1989 ANSI C (C89) was established, standardizing the language for the first time.
1999 C99 standard was released, introducing new features like inline functions and variable-length arrays.
2011 C11 standard was released, adding features for multi-threading and improved Unicode support.
2018 C18 standard was released, which was a bug fix of C11 without any new features.
Today, C is one of the most widely used programming languages. It has influenced many other languages, including C++, C#, Objective-C, and even Java. It continues to be used for system programming, game development, and in embedded systems.

  • 1989: ANSI C (C89) was established, standardizing the language for the first time.
  • 1999: C99 standard was released, introducing new features like inline functions and variable-length arrays.
  • 2011: C11 standard was released, adding features for multi-threading and improved Unicode support.
  • 2018: C18 standard was released, which was a bug fix of C11 without any new features.
